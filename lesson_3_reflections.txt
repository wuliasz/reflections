When would you want to use a remote repository rather than keeping all your work local?

when working with others on a project.
when working with more than one machine. 


Why might you want to always pull changes manually rather than having Git 
automatically stay up-to-date with your remote repository?

automatic pull may affect local changes

Describe the differences between forks, clones, and branches. 
When would you use one instead of another?

forks are copies of entire repositories that exist only on the github site.
clones are copies of entire repositories that can be copied from or to any machine.
branches are only segments of repositories.
